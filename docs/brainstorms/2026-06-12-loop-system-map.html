The Loop System

The developer authors loops; the system runs them; the human holds the corpus gate. An edge graduates from manual to automatic only by passing its pre-committed signal gate (ADR 0001). Default mode prompts the human at every judgment point; autonomy is a per-run, explicit parameter — never an inheritance.

Loop 01 — Learning-capture

Every durable learning from merged work reaches the corpus without relying on anyone's memory.

Loop 02 — Drift

Plans and reality reconverge: drift is measured per run, aggregated across runs, and read back into planning.

Loop 03 — Review

A change converges to clean — findings fixed, reviewer comments triaged and resolved — without the human driving each pass.
